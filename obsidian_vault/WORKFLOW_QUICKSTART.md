# Kimi × Codex 工作流 - 快速啟動

> 複製以下結構即可開始新專案

---

## 1. 建立目錄結構

```bash
mkdir -p obsidian_vault/reviews
```

---

---

## 3. 開始協作流程

```
Kimi: 分析 → 撰寫 Prompt → 覆寫 codex_prompt.md
                      ↓
Codex: 讀取 → 實作 → 更新任務狀態
                      ↓
Kimi: 驗收 → 建立 review → 更新進度
                      ↓
下一個任務...
```

---

## 5. 常用指令
